# 🎮 VGC Team Space

> **VGC Team Space** é uma aplicação web projetada para ajudar jogadores competitivos de Pokémon (formato VGC) a gerenciar e organizar seus times de forma eficiente, sem conflitos de Pokémon duplicados.

---

## 🎯 O Problema & A Solução

No cenário competitivo de Pokémon VGC, jogadores frequentemente constroem e testam múltiplos times. No entanto, na prática (especialmente se você joga nos jogos oficiais do console como *Pokémon Scarlet & Violet*), você pode ter apenas **um** exemplar treinado de um determinado Pokémon (ex: um único *Gholdengo* ou um único *Ogerpon-Hearthflame* competitivo). 

Se você tentar usar o mesmo Pokémon em múltiplos times de torneio ao mesmo tempo, pode gerar conflitos de disponibilidade no console.

O **VGC Team Space** resolve isso:
- **Armazenamento de Times:** Salve todos os seus times competitivos em um só lugar.
- **Organização sem Conflitos:** Disponibiliza formas de organizar e planejar o uso de seus times de modo a destacar e evitar conflitos caso você não tenha múltiplos exemplares do mesmo Pokémon disponíveis fisicamente no seu jogo.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias modernas de desenvolvimento web:

- **React** (Biblioteca para interfaces de usuário)
- **TypeScript** (Tipagem estática para robustez do código)
- **Vite** (Ferramenta de build extremamente rápida)
- **CSS Vanilla** (Estilização responsiva e customizada)

---

## 🚀 Como Rodar o Projeto Localmente

1. **Instale as dependências:**
   ```bash
   npm install
   ```

2. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Gere a build de produção:**
   ```bash
   npm run build
   ```
